# Settings

To configure the Shopify Taxonomy module settings:

1. Click **Settings** in the main menu.
1. In the search field of the next blade, type **Shopify** to find the settings related to the module.
1. Click **General**.
1. In the next blade, configure the following settings:

    ![Settings](media/settings.png){: style="display: block; margin: 0 auto;" }<br><br>
    
1. Click **Save** in the toolbar to save the changes.

The settings have been saved.

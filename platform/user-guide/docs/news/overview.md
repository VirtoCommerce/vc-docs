# Overview

The **News** module enables content managers to publish news articles for selected user groups using different languages within the Virto Commerce Platform. Additionally, the module offers features for scheduling articles publishing.

[![Source code](media/source_code.png)](https://github.com/VirtoCommerce/vc-module-news)

[![Latest release](media/latest_release.png)](https://github.com/VirtoCommerce/vc-module-news/releases)

## Key features

The diagram below illustrates the connections of the News module:

![Key entities](media/key-entities.png){: style="display: block; margin: 0 auto;" }


With the News module, users can:

* Write articles using extended markup.
* Localize articles into multiple languages.
* Select multiple target user groups.
* Schedule publication.
* Clone articles.
* Set up SEO data.

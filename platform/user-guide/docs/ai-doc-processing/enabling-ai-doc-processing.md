# Enable AI Document Processing

To enable AI document processing:

1. Click **Settings** in the main menu.
1. In the next blade, select settings related to the **AI Document Processing** module.
1. Click **Purchase request**.
1. In the next blade, switch the **Enable purchase requests** option to on.
1. Click **Save** in the toolbar to save the changes.

Now, you users can download various types of bulk orders.

![AI enabling](media/enabling-doc-processing.gif)

You can [configure other settings](settings.md).

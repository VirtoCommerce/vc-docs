# Overview

The **Application Insights** module enables the collection of key metrics, application telemetry, and trace logging data, providing deep insights into the performance and health of your application within Microsoft Azure.

![App Insights dashboard](media/dashboard.png){: style="display: block; margin: 0 auto;" }

[![Source code](media/source_code.png)](https://github.com/VirtoCommerce/vc-module-app-insights)

[![Latest release](media/latest_release.png)](https://github.com/VirtoCommerce/vc-module-app-insights/releases)

## Key features

With the Application Insights, users can:

* Collect standard metrics.
* Collect application telemetry data.
* Collect application trace logging data.
* Offer flexible configuration through settings or code.
* Allow configuration of Application Insights settings, including specifying the **Instrumentation key** for each store.

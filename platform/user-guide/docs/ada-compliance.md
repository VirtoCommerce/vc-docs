# ADA Compliance

Virto Commerce maintains active compliance with the Americans with Disabilities Act (ADA) and the Web Content Accessibility Guidelines (WCAG). Ongoing enhancements aligned with the latest standards are released regularly as part of Virto Commerce Platform updates. This reduces legal risk for enterprises operating in regulated markets.

Virto Commerce includes accessibility features by default, so you can deliver an inclusive shopping experience to every customer with no additional configuration. The Platform and Frontend ship with built-in:

* **Keyboard navigation**: All interactive elements support full keyboard control. Navigation menus, product selectors, and checkout forms can be operated without a mouse using standard shortcuts.
* **ARIA support**: Virto Commerce uses Accessible Rich Internet Applications (ARIA) labels and semantic HTML to support screen readers. Form fields, navigation elements, and dynamic content are compatible with screen readers. Tested screen readers include JAWS, NVDA, and VoiceOver.
* **Zoom, high-contrast color, and scaling**: Zoom and scaling are always available for users with visual impairments. They cannot be disabled on the core pages (Login, Catalog, and Cart). Users can adjust content size without breaking layout or functionality.
* **Responsive layouts** that align with the Americans with Disabilities Act (ADA) and Web Content Accessibility Guidelines (WCAG). The Frontend is fully responsive. Accessibility is preserved across desktop, mobile, and tablet devices. Touch targets meet minimum size requirements so usability remains consistent on smaller screens.

# Configure Tax Providers   

Virto Commerce allows you to configure tax providers based on your business needs. By default, the [Tax module](/platform/user-guide/latest/tax/overview) includes the **Fixed rate tax provider**. If you require automated tax calculations, you can install the [Avalara tax module](/platform/user-guide/latest/integrations/avalara/overview) to use the **Avalara** tax provider.

# PromotionCouponType ==~object~==

This type represents a promotion coupon.

## Fields

| Field                   | Description                                    |
|-------------------------|------------------------------------------------|
| `id`  ==String!==       | The Id of the promotion coupon.                |
| `endDate`  ==DateTime== | The expiration date of the promotion coupon.   |
| `systemName`  ==String==| The system name of the promotion.              |
| `label`  ==String==     | The localized label of the promotion.          |
| `name`  ==String==      | The localized name of the promotion.           |
| `description`  ==String==| The localized description of the promotion.  |
| `couponCode`  ==String==| The coupon code of the promotion.              |

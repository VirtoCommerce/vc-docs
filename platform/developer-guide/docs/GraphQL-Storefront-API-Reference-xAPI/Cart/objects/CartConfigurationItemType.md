# CartConfigurationItemType ==~object~==

This type represents configuration items for configurable product.

## Fields

| Field        | Description                                                         |
|--------------|---------------------------------------------------------------------|
| `id` ==String!== | The Id of the cart configuration item.              |
| `name` ==String== | The name of the cart configuration item.                          |
| `sectionId` ==String== | The Id of the section to which the configuration item belongs. |
| `productId` ==String== | The Id of the product associated with the configuration item. |
| `quantity` ==Int== | The quantity of the product in the configuration item.            |

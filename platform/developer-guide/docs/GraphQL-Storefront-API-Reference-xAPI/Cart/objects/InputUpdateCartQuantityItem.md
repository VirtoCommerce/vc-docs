# InputUpdateCartQuantityItem ==~object~==

This type represents an item in the shopping cart for which the quantity needs to be updated.

## Fields

| Field                   | Description                                        |
| ----------------------- | -------------------------------------------------- |
| `productId` ==String!== | The Id of the product to update.    |
| `quantity` ==Int!==     | The new quantity to set for the specified product. |

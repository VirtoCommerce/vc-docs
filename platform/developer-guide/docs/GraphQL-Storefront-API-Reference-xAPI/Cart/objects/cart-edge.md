# CartEdge ==~object~==

The `CartEdge` type represents an edge in a connection of carts.

## Fields

| Field                                        | Description                               |
|----------------------------------------------|-------------------------------------------|
| `cursor` ==String==                         | A string representing the cursor for this edge. |
| `node` [==CartType==](cart-type.md)        | The `CartType` object associated with this edge. |

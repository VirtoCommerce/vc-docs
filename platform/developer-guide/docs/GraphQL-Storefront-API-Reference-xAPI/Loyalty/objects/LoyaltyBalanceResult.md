# LoyaltyBalanceResult ==~object~==

This type defines the structure of a loyalty balance result and contains information about the current and resulting balances after an operation.

## Fields

| Field                         | Description                                                                |
| ----------------------------- | -------------------------------------------------------------------------- |
| `currentBalance` ==Decimal!== | The current balance of the loyalty account before the operation.           |
| `resultBalance` ==Decimal!==  | The resulting balance of the loyalty account after applying the operation. |

# LoyaltyOperationLogObject ==~object~==

This type represents the object associated with a loyalty program operation log entry, such as an order.

## Fields

| Field                    | Description                                                               |
| ------------------------ | ------------------------------------------------------------------------- |
| `type` ==String!==       | The type of the object linked to the loyalty operation.     |
| `orderId` ==String==     | The Id of the order associated with the operation, if any. |
| `orderNumber` ==String== | The number of the order associated with the operation, if any.            |

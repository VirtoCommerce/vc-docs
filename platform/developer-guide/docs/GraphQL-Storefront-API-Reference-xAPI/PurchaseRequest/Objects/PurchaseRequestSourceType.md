# PurchaseRequestSourceType ==~object~==  

This type represents a source associated with a purchase request, including its metadata and details.  

## Fields  

| Field                     | Description                                                       |  
|---------------------------|-------------------------------------------------------------------|  
| `name` ==String!==        | The name of the source file.                                      |  
| `url` ==String!==         | The URL where the source file can be accessed.                    |  
| `contentType` ==String!== | The type of the source file.                                      |  
| `size` ==Long!==          | The size of the source file in bytes.                             |

# PageDocumentType ==~object~==  

This type represents a page document stored in the Pages module.  

## Fields  

| Field          | Description                                      |  
|-----------------|--------------------------------------------------|  
| `id` ==String!==        | The unique Id of the page document.   |  
| `source` ==String==      | The source CMS of the page document (e.g., Builder.io). |  
| `permalink` ==String==   | The URL-friendly Id for the page.     |  
| `content` ==String!==    | The main content of the page document.        |

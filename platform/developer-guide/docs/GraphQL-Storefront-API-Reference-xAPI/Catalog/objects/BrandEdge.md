# BrandEdge  ==~object~==

This type represents an edge that connects a brand to a connection.

## Fields

| Field                                | Description                                                           |
| ------------------------------------ | --------------------------------------------------------------------- |
| `cursor` ==String!==                 | A cursor that represents the position of the brand in the connection. |
| `node` [==BrandType==](BrandType.md) | The brand node associated with the edge.                              |

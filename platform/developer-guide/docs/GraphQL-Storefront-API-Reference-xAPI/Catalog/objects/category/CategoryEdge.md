# CategoryEdge ==~object~==

This type represents a connection between a node (a category) and its associated cursor. 

## Fields

| Field                  	| Description                                   	|
|------------------------	|-----------------------------------------------	|
| `cursor`  ==String!==  	| The cursor associated with the category node. 	|
| `node`  ==Category==   	| The category associated with the edge.        	|

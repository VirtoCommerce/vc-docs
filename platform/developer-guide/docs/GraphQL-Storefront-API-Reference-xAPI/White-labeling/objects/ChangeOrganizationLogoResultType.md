# ChangeOrganizationLogoResultType ==~object~==

This type defines the result of an attempt to change an organization's logo.

## Fields

| Field                      | Description                                                                       |
|----------------------------|-----------------------------------------------------------------------------------|
| `isSuccess` ==Boolean!==   | Indicates whether the operation to change the organization's logo was successful. |
| `errorMessage` ==String!== | A message providing details of the error, if the operation was unsuccessful.      |

# FaviconType ==~object~==

This type defines the structure of a favicon used in a web page.

## Fields

| Field               | Description                                                       |
|---------------------|-------------------------------------------------------------------|
| `rel` ==String!==   | The relationship between the current document and the favicon.    |
| `type` ==String!==  | The type of the favicon.                                          |
| `sizes` ==String!== | The sizes of the favicon.                                         |
| `href` ==String!==  | The URL of the favicon file, pointing to its location on the web. |

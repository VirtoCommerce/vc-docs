# ProductPickupAvailabilityType ==~enum~==

This enum represents the availability status of a product at a pickup location, indicating how soon the product can be picked up.

## Values

| Value            | Description                                                           |
| ---------------- | --------------------------------------------------------------------- |
| `Today`          | The product is available today, typically within hours.               |
| `Transfer`       | The product is available via transfer, typically within days.         |
| `GlobalTransfer` | The product is available via global transfer, typically within weeks. |

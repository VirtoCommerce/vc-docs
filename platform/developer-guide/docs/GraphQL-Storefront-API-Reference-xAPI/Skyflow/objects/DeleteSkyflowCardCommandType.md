# DeleteSkyflowCardCommandType ==~object~==

This type represents a command to delete a stored Skyflow card.

## Fields

| Field                   | Description                                          |
| ----------------------- | ---------------------------------------------------- |
| `skyflowId` ==String!== | The Id of the Skyflow card to delete.                |
| `storeId` ==String!==   | The Id of the store associated with the card.        |

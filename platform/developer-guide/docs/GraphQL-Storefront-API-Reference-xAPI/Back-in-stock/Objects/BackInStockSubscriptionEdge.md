# BackInStockSubscriptionEdge ==~object~==  

This type represents an edge in a paginated list of back-in-stock subscriptions.  

## Fields  

| Field                     | Description                                      |  
|---------------------------|--------------------------------------------------|  
| `cursor` ==String!==       | A cursor for pagination, pointing to this edge. |  
| `node` [==BackInStockSubscriptionType==](BackInStockSubscriptionType.md) | The back-in-stock subscription at this edge. |

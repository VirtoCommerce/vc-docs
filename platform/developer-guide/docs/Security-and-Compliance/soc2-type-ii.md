# SOC 2 Type II Compliance

Virto Cloud holds an active SOC 2 Type II certification for the 2025 audit period. This is the third consecutive year that Virto Cloud has been certified.

SOC 2 is an auditing procedure defined by the American Institute of Certified Public Accountants (AICPA). It evaluates how a cloud service provider designs and operates controls aligned with the AICPA Trust Services Criteria: security, availability, processing integrity, confidentiality, and privacy.

A SOC 2 report can be issued in two forms:

- Type I evaluates the design of controls at a single point in time.
- Type II is the type Virto Commerce has been certified for. It evaluates both the design and the operating effectiveness of controls over an extended period, typically several months.

The SOC 2 Type II report provides independent evidence that:

- Virto Cloud operates security and availability controls aligned with the AICPA Trust Services Criteria.
- Those controls operated effectively across the full audit period.
- The report can be supplied as input to customer-side audits, vendor risk reviews, and procurement workflows.

# Overview

The following folders with files define the Frontend functionality, appearance and texts:

| Folder                | Function                      | Content                                                                                                |
|-----------------------| ------------------------------|----------------------------------------------------------------------------------------------	|
| **client-app/assets** | [Visual theme customization](visual-theme-customization.md)    | <ul><li>**icons**: SVG and icon files used throughout the Frontend.</li><li>**images**: Static images such as banners, logos, and placeholders.</li><li>[presets](visual-theme-customization.md#presets): Predefined configurations for the theme.</li><li>[styles](visual-theme-customization.md#styles-customization): SCSS/CSS files that define the look and feel of the Frontend, including colors, spacing, typography, and component styles.</li></ul> |
| **config**    	    | [Functionality customization](functionality-customization.md)   |<ul><li>[manifest.webmanifest](functionality-customization.md#manifestwebmanifest): Metadata for the web application.</li><li>[menu.json](functionality-customization.md#menujson): The navigation menu structure and items for the Frontend.</li> <li>[product-default.json](functionality-customization.md#product-defaultjson): Default product display settings.</li> <li>[product_b2c.json](functionality-customization.md#product_b2cjson): Specific product configuration for B2C stores.</li> <li>[settings_data.json](functionality-customization.md#settings_datajson): Feature configurations. You can enable/disable quotes and product comparison, set limits for lists, configure order and quote statuses, and more.</li></ul> |
| **locales**   	    | [Texts customization](texts-customization.md)           | Locale files used to provide translated content for the Frontend, including labels, messages, and UI text.                                             	|

![Readmore](media/readmore.png){: width="25"} [Complete Frontend structure](https://github.com/VirtoCommerce/vc-frontend/blob/dev/README.md#the-application-structure)

Some components are intuitive, while others can be clicked to view a detailed customization guide.

# Saved for Later

The **Saved for later** section stores all the product the user leaves for later purchase. Here you can:

* Add all the listed items to cart with a single click.
* Add specific listed items to cart.
* Update the amount of items in cart.
* Delete items from the list.
* Purchase items separately from cart. This allows customers to manage and purchase items from a list without affecting their shopping cart.

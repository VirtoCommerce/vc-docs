# Save for Later

The **Save for later** feature allows customers to move items from their shopping cart into a dedicated list for future purchase without losing track of them.

With this feature, customers can:

* Save products from the cart for later consideration.
* View all saved items in a dedicated **Saved for later** widget on the cart page.
* Access a **Saved for later** list at any time.
* Easily move saved products back into the cart when ready to buy.

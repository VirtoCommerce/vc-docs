# Brands Page

For easier navigation, users can open the [Brands page](https://virtostart-demo-store.govirto.com/brands), where all available brands are listed in alphabetical order:

![Brands page](../media/brands-page.png){: style="display: block; margin: 0 auto;" }

Clicking a brand logo or name opens a dedicated brand page displaying all products (as product cards) associated with that brand:

![Brand products](../media/brand-products.png){: style="display: block; margin: 0 auto;" width="500"}

Here, users can:

* Sort products by various properties.
* Filter products by:
    * **Purchased before** status.
    * **In stock** status.
    * Availability at specific branches. 
* Add products to cart.

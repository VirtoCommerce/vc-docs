# CancelQuoteCommandType ==~object~==

The `CancelQuoteCommandType` is used to cancel a quote. 

## Fields

| Field                   | Description                         |
| ----------------------- | ------------------------------------|
| `quoteId` {==String!==} | The Id of the quote to be canceled. |
| `comment` {==String!==} | A comment for canceling the quote.  |
